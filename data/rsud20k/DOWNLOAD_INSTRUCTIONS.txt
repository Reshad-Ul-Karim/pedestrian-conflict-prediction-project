RSUD20K Dataset Download Instructions
============================================================

To download the full RSUD20K dataset:

1. Clone the repository:
   git clone https://github.com/hasibzunair/RSUD20K.git data/rsud20k/raw

2. Or download from the official source:
   Visit: https://github.com/hasibzunair/RSUD20K

3. For partial download, use git sparse-checkout:
   cd data/rsud20k/raw
   git init
   git config core.sparseCheckout true
   echo 'images/*' >> .git/info/sparse-checkout
   echo 'annotations/*' >> .git/info/sparse-checkout
   git remote add origin {self.repo_url}.git
   git pull origin main

After downloading, run this script again to process the data.
